### Описание
Консольное приложение выводит список галактик с их названиями, расстоянием (в мегасветовых годах) и морфологическим типом. В исходном коде содержались две ошибки, которые были обнаружены и исправлены в процессе отладки.

### Обнаруженные и исправленные ошибки

**Ошибка №1 — неверный тип свойства:**
```csharp
// Было:
object GalaxyType;

// Исправлено:
GType GalaxyType;
```

**Ошибка №2 — опечатка в символе в операторе `case`:**
```csharp
// Было:
case 'l':

// Исправлено:
case 'I':
```

### Использованные способы отладки

| Инструмент | Описание применения |
|------------|----------------------|
| **Точки останова (Breakpoints)** | Установлены на строке `Console.WriteLine` внутри цикла `foreach` и в конструкторе `GType`. |
| **Подсказки по данным (Data Tips)** | При наведении на `theGalaxy.GalaxyType` и последующем раскрытии обнаружено поле `MyGType`. |
| **Шаг с заходом (Step Into, F11)** | Выполнен вход в конструктор `GType` для проверки работы оператора `switch`. |
| **Правка и продолжение (Edit and Continue)** | Исправление обращения `GalaxyType.MyGType` непосредственно во время отладочной сессии. |
| **Перезапуск (Restart, Ctrl+Shift+F5)** | Повторный запуск приложения с отладчиком после исправления типа свойства. |

## Задание 3 — Приложение «Буквы»

### Результат выполнения
<img width="550" height="414" alt="image" src="https://github.com/user-attachments/assets/e527b995-2cc4-4f7d-8409-e27f865d7f3a" />

### Описание
Консольное приложение выполняет итерацию по массиву символов `letters`. На каждом шаге формируется строка `name`, после чего вызывается метод `SendMessage`. Программа служит обучающим примером для знакомства с основными инструментами отладчика.

### Использованные способы отладки

| Инструмент | Описание применения |
|------------|----------------------|
| **Точки останова (Breakpoints)** | Установлены на строке `name += letters[i]` внутри цикла `for`. |
| **Подсказки по данным (Data Tips)** | Раскрытие массива `letters[10]` для просмотра всех элементов; отслеживание значения `name` на каждой итерации. |
| **Выполнить до щелчка (Run to Click)** | Быстрый переход к строке `Console.WriteLine` внутри метода `SendMessage`. |
| **Автоматические переменные (Autos)** | Автоматическое отображение переменных, используемых в текущей и предыдущей строках. |
| **Локальные переменные (Locals)** | Просмотр переменных `letters`, `name`, `a`, `i` в текущей области видимости. |
| **Контрольные значения (Watch)** | Добавлена переменная `name` для наблюдения за её ростом по итерациям. |
| **Стек вызовов (Call Stack)** | Просмотр цепочки вызовов `Main → RunLetters → SendMessage`; двойной щелчок для навигации по коду. |
| **Перезапуск (Restart)** | Повторный запуск с начала для проверки поведения программы при различных точках останова. |
